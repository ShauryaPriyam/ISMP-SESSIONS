What is HTML?

HTML stands for Hyper Text Markup Language. It is used to structure content on the web, including text, links, images, and more.

HTML elements form the building blocks of all websites. Every web page you visit is built using HTML.

- Shaurya Priyam

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout web pages — for example, to change fonts, colors, spacing, or layouts.

CSS can be internal, external, or inline. It works alongside HTML to create visually appealing websites.

- Shaurya Priyam
